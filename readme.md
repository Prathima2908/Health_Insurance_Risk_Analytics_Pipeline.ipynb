# Health Insurance Risk & Cost Analytics Pipeline

## Overview

This project builds an end-to-end analytics pipeline to analyze customer risk, claim behavior, and revenue trends in a health insurance dataset. It combines data engineering, analytics, and machine learning techniques to generate actionable insights.

---

## Key Features

* Data cleaning and validation
* Feature engineering (risk score, claim ratio, segmentation)
* KPI analysis (premium vs claims, profitability)
* Customer segmentation using K-Means clustering
* Time-based trend analysis
* Time-series forecasting using ARIMA
* Data visualization for insights

---

## Tech Stack

* Python (Pandas, NumPy)
* Visualization (Matplotlib, Seaborn)
* Machine Learning (Scikit-learn)
* Time Series (Statsmodels)
* Google Colab

---

## Dataset

Synthetic health insurance dataset including:

* Demographics (age, gender, region)
* Health metrics (BMI, smoking status, chronic disease)
* Financial data (premium, claims, profit)
* Behavioral attributes

---

## Key Analysis

### Risk Analysis

Calculated a custom risk score based on health and behavioral factors.

### Customer Segmentation

Applied K-Means clustering to group customers based on risk and financial metrics.

### KPI Insights

Analyzed premium collection, claims, and profitability across segments.

### Forecasting

Built ARIMA model to predict future premium trends.

---

## Business Impact

* Identifies high-risk customers
* Improves pricing and underwriting decisions
* Tracks profitability across policies
* Supports financial forecasting
